# WINDOWS SHORT-CUT KEYS

```
            alt + tab  -->  To swich the programm
            alt + spacebar  ---> Restore/move/minimise/close the page
            alt + esc      ---> Switch the programm
            alt + F4       ---> Close the programm
            alt + F4 + Enter ----> shut down the windows

            ctrl + right/left arrow --->Move word by word
            ctrl + shift + esc ---> Open the task manager
            ctrl + up/down arrow ---> move to up/down in the paragraph
            ctrl + esc ---> open the start menu
            ctrl + F4 ---> close thye programm
            F1  ----> open the help window of that programm

            windows + L ---> lock the Screen
            windows + pause Break --> Properties of the system
            windows + E ---> open File-Explorer
            windows + M ---> Minimise the Windows
            windows + shift + M ---> Maximise the Windows 
            windows + D ---> Minimise all the windows
            windows + right/left arrow ---> Shift window in right/left

            ctrl + insert key ---> copy the content
            shift + insert key ---> paste the content

            home key ---> Move to 1st word of 1st line in  that page
            shift + home --->Select the line (right to left)
            ctrl + shift + home --->Select the paragraph

            End ----> it works opposite to Home key

            pg up/down ---> to scroll the page

            delete key ---> it works oppsite the backspace key
            ctrl + delete ---> Delete the word

            F1 ----> help
            F2 ----> Rename the File
            alt + ctrl + F2 ----> Search the File
            ctrl + F2 ---> Print Review

            F3 ---> Searching
            shift + F3 ---->Change the Case( upper to lower and vice-versa)

            F4 ---> go to addressBar
            alt + F4 ---> Close the Windows 

            F5 ----> Refresh the Windows
            F6 ----> got to addressBar
            ctrl + shift + F6 ----> Review 2 files one by one in ms-word and Excel

            F7 ----> to check spelling in the word
            shift + F7 ---> to search related word

            F8 ----> works in booting the OS 
            F9 ----> to refresh the file in ms-word or open bios menu in starting the window

            F10 ---> open the menuBar

            F11 ----> Resume the previous window in chrome/firefox/....

            F12 ---> Save the file
            ctrl + F12 ---> open new file in MS-word
            shift + F12 ----> Print the file
```

# Mouse SHORT-CUT KEYS

```
            alt + shift + numLock ---> open the mouse menu
            8 ----> cursor up
            2 ----> cursor down
            4 ----> cursor lefty
            6 ----> cursor right
            7 ----> cursor up and left
            9 ----> cursor up and right
            3 ----> cursor down and right
            1 ----> cursor down and left

            / ----> works like left button of  the Mouse
            * ----> works like right button of the Mouse
            5 ----> select the file
            5 ----> (On Double type) open the file
            0 ----> Lock the folder/file for drag and drop
```
